SMART STUDY PLANNER                       

Description
Smart Study Planner is a Django-based web application that helps students organize their study schedule, manage tasks, and plan their learning activities efficiently.

The goal of this project is to provide a simple and user-friendly platform for students to manage their daily study plans.

Features

- Create and manage study tasks
- Organize daily study schedules
- User-friendly interface
- Django based backend
- Database management using SQLite
- Responsive web pages
 
Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite
- Bootstrap

Installation & Setup

Follow these steps to run the project locally:
 1. Clone the repository :
     git clone https://github.com/samarkumarbaronia/Smart-Study-Planner.git

 2. Go inside the project folder : cd Smart-Study-Planner

 3. Create virtual environment :  python -m venv studyenv

 4. Activate virtual environment : studyenv\Scripts\activate

 5. Install dependencies : pip install -r requirements.txt

 6. Run migrations : python manage.py migrate
 
 7. Start server : python manage.py runserver
    Open browser : http://127.0.0.1:8000/
